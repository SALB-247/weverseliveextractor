Weverse 1080p URL 추출기

playInfo 응답을 붙여넣으면 1080p mp4 주소를 뽑아줍니다. 네트워크 가로채기 없이, 붙여넣은 텍스트만 파싱합니다.

쓰는 법

  1. Weverse 영상 페이지에서 영상을 재생
  2. F12 → Network 탭 → 필터창에 playInfo 입력
  3. 목록에서 Type이 xhr인 행을 선택 (자세히는 아래 주의 참고)
  4. 그 행을 우클릭 → Copy → Copy response
  5. 아래 칸에 붙여넣고 URL 추출 클릭
  6. 나온 카드의 복사 버튼으로 1080p mp4 주소 복사

⚠ 주의할 점

  • preflight 행은 고르지 마세요. playInfo로 필터하면 같은 요청이 여러 줄 뜨는데, Type이 preflight인 행은 CORS 사전 요청이라 응답 본문이 비어 있습니다. 반드시 Type이 xhr인 행을 고르세요. (둘이 헷갈리면 둘 다 Copy response 해보고 <MPD가 들어간 쪽을 쓰면 됩니다.)
  • 영상을 재생해야 요청이 발생합니다. 페이지 진입만으로는 playInfo가 안 잡힐 수 있으니, 잡히지 않으면 영상을 한 번 재생한 뒤 다시 확인하세요.
  • 서명은 곧 만료됩니다. 응답 XML 안의 expireTime 시각이 지나면 URL 끝 _lsu_sa_ 서명이 무효가 되어 더 이상 동작하지 않습니다. 만료되면 페이지에서 응답을 새로 복사해 다시 붙여넣으세요. (추출 결과 카드에 만료 시각이 함께 표시됩니다.)
  • 본편과 오디오 필터 버전이 둘 다 나올 수 있습니다. 응답에 playback(본편)과 enhanceModePlaybacks(오디오 필터)가 모두 있으면 각각 배지로 구분해 표시됩니다. 보통 필요한 건 본편입니다.
